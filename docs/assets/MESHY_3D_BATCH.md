# Meshy.ai 3D Model Generation Batch

## Instructions
1. Go to [Meshy.ai](https://www.meshy.ai)
2. Select "Text to 3D" or "Image to 3D"
3. Copy each prompt below
4. Settings: "Game Asset" style, Medium/High quality
5. Export as GLB or FBX
6. Naming: Use the ID (e.g., F01, W01) as filename prefix

## Global Style Modifiers
Add to ALL prompts for consistency:
```
, low-poly stylized medieval fantasy, game-ready asset, clean topology, soft shadows, PBR textures, white background
```

# PRODUCTION NOTES

## Batch Processing Tips
1. Generate 3-5 variants of each prompt, select best
2. Process by category for visual consistency
3. Name files with ID prefix immediately: `F01_stone_foundation_1x1.glb`
4. Keep rejected generations - may be useful as variants

## Post-Processing Checklist
- [ ] Check polycount (target 500-2000 tris)
- [ ] Verify origin point (center bottom)
- [ ] Confirm scale (1 unit = 1 meter)
- [ ] Test UV mapping
- [ ] Apply consistent naming
