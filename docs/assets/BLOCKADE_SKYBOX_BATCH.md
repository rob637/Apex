# PRODUCTION NOTES

## Blockade Labs Settings
- Select "Fantasy" or "Digital Painting" style
- Set resolution to maximum (8192x4096)
- Generate in HDR format when possible
- Enable "Enhance" for better quality

## Download Options
- **HDR**: Best for Unity HDRP projects
- **EXR**: Good for high dynamic range
- **PNG Equirectangular**: Standard format for all engines

## Unity Skybox Setup
```csharp
// Create Skybox Material
Material skyboxMaterial = new Material(Shader.Find("Skybox/Panoramic"));
skyboxMaterial.SetTexture("_MainTex", skyboxTexture);
skyboxMaterial.SetFloat("_Rotation", 0);
skyboxMaterial.SetFloat("_Exposure", 1.0f);
RenderSettings.skybox = skyboxMaterial;
```

## Unity Import Settings
```
Texture Type: Default
Texture Shape: Cube
Mapping: Latitude-Longitude Layout (for equirectangular)
Convolution Type: None
Filter Mode: Trilinear
Compression: High Quality
Max Size: 4096 or 8192
```

## Time of Day System Integration
```csharp
// Blend between skyboxes for smooth transitions
[SerializeField] Material daySkybox;
[SerializeField] Material nightSkybox;

void UpdateSkybox(float timeOfDay) {
    // timeOfDay: 0 = midnight, 0.5 = noon, 1 = midnight
    float blend = CalculateSkyboxBlend(timeOfDay);
    RenderSettings.skybox.Lerp(nightSkybox, daySkybox, blend);
}
```
