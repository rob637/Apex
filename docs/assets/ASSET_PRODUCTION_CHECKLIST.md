```
    ├── Combat/
    ├── Building/
    └── Environment/
```

## Step 3: Generation Pipeline
1. **Batch Generate** - Run all prompts in batches
2. **Review & Select** - Choose best outputs
3. **Post-Process** - Clean up, resize, optimize
4. **Import to Unity** - Configure settings
5. **Test in Game** - Verify appearance
6. **Iterate** - Regenerate if needed

## Step 4: Quality Control
- [ ] All assets follow naming convention
- [ ] Consistent style across all assets
- [ ] File sizes optimized for mobile
- [ ] All required variants present
- [ ] Unity import settings correct
